# AVL Tree Collection (MyCollection<T>)

Реализация самобалансирующегося бинарного дерева поиска (АВЛ-дерево) на языке C#. Проект выполнен в рамках учебного курса по структурам и алгоритмам обработки данных.

## Основные возможности:
* **Балансировка:** Автоматическая балансировка дерева при добавлении и удалении узлов (высота дерева $O(\log n)$).
* **Интерфейсы .NET:** Полная реализация интерфейсов `ICollection<T>`, `IEnumerable<T>` и `ICloneable`.
* **Глубокое копирование:** Реализован механизм клонирования коллекции с полным копированием узлов.
* **Иерархия объектов:** Работа с базовым классом `Product` и его наследниками (`Toy`, `DairyProduct`, `FoodProduct`).

## Особенности реализации:
* Обобщенный тип данных с ограничением `where T : IComparable<T>`.
* Двухуровневая логика сравнения объектов (по названию и цене).
* Итератор (`yield return`) для обхода дерева в порядке сортировки (In-Order).

## Стек технологий:
* C#, .NET 6.0/8.0
* Visual Studio 2022
